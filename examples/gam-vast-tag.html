Google Ad Manager → TNE Catalyst VAST Tag

Drop the tag below into the Google Ad Manager line item / creative as a Third Party VAST Redirect. All publisher-supplied enrichments (page, referrer, cache buster, ad unit, content key-values, privacy signals) are forwarded to /video/vast via GAM macros and mapped server-side onto the OpenRTB 2.5 bid request Catalyst emits to demand.

Replace before use:
  • {PUBLISHER_ID} — your Catalyst publisher ID (e.g. pub-video-123456).
  • {GVL_ID} in %%GDPR_CONSENT_{GVL_ID}%% — Catalyst’s TCF vendor (GVL) ID. Ask your Catalyst account manager for the current value.
  • {MIN}/{MAX} — creative duration constraints in seconds.
  • {FLOOR} — bid floor in USD (omit if not needed).

Production VAST Tag URL

https://api.tne-catalyst.com/video/vast?pub_id={PUBLISHER_ID}&w=%%WIDTH%%&h=%%HEIGHT%%&mindur={MIN}&maxdur={MAX}&mimes=video/mp4,application/x-mpegURL&protocols=2,3,5,6,7,8&placement=1&linearity=1&bidfloor={FLOOR}&placement_id=%%ADUNIT%%&page_url=%%PAGE_URL%%&domain=%%SITE_DOMAIN%%&ref=%%REFERRER_URL%%&description_url=%%DESCRIPTION_URL%%&cb=%%CACHEBUSTER%%&sport=%%PATTERN:sport%%&competition=%%PATTERN:competition%%&lang=%%PATTERN:language%%&device_type=%%PATTERN:device%%&geo=%%PATTERN:geo%%&content_type=%%PATTERN:content_type%%&gdpr=%%GDPR%%&gdpr_consent=%%GDPR_CONSENT_{GVL_ID}%%&addtl_consent=%%ADDTL_CONSENT%%&us_privacy=%%US_PRIVACY%%&gpp=%%GPP_STRING%%&gpp_sid=%%GPP_SID%%&coppa=%%TFCD%%&ifa=%%ADVERTISING_IDENTIFIER_PLAIN%%&ifa_type=%%ADVERTISING_IDENTIFIER_TYPE%%&lmt=%%LIMITADTRACKING%%&ua=%%USER_AGENT_ESC%%&session_id=%%CLICK_ID%%&schain=%%SCHAIN%%

GAM substitutes each %%...%% at ad call time, so the request that hits Catalyst is fully populated. Line breaks above are for readability only — the tag is a single URL.

Custom Key-Values (Contextual Targeting)

Each KV defined in your Ad Manager network is read via the %%PATTERN:<key>%% macro and forwarded as a typed query parameter. Catalyst attaches them to the bid request under site.content and imp[0].ext.context so downstream DSPs can target them directly.

Testing endpoint: Use https://test.tne-catalyst.com/video/vast?... (same query string) against publisher ID pub-video-test before going live. Once the first ~1,000 bid requests land, the Catalyst team will verify that every enrichment is being populated (no %%…%% strings passing through) and share a troubleshooting report.

Where to paste this in Google Ad Manager

  1. GAM UI → Delivery → Orders → new or existing order.
  2. Create a Line item with type Price priority (or Sponsorship for guaranteed).
  3. Set Creative type = Video and choose VAST redirect.
  4. Paste the URL above into the VAST Tag URL field.
  5. Enable “Convert to XML” off — Catalyst already returns VAST XML.
  6. Save, approve, and traffic.
